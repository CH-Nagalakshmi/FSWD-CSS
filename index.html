pointers for div

  1. width of a block level element is 100% of the viewport width
  2. height of a block isn`t set by default
  3. height and width can be explicitly set using CSS
POINTERS FOR INLINE
  1. width of a inline level element is dependent on the content
  2. height of a inline isn`t set by default
  3. height and width can not be explicitly set using CSS
